This is the code to generate results published in the paper "Machine Learning Link Inference of Noisy Delay-Coupled Networks with Optoelectronic Experimental Tests", by Amitava Banerjee, Joseph D. Hart, Rajarshi Roy, and Edward Ott, Phys. Rev. X 11, 031014 â€“ Published 20 July 2021

Contact Amitava Banerjee at amitava8196@gmail.com or amitavab@umd.edu for questions or comments.

The code has two parts. The first part simulates a 4-node optoelectronic oscillator network with a specified network topology and a given set of parameter values in our paper and the second part infers the network with reservoir computing.
